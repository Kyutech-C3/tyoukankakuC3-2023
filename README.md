## コマンド

### 起動

```sh
docker compose up -d
```

### ダウン

```sh
docker compose down
```

ボリュームを削除する場合

```sh
docker compose down -v
```

### ストップ

```sh
docker compose stop
```

### スタート

```sh
docker compose start
```

### 再起動

```sh
docker compose restart
```

### ビルド

```sh
docker compose build
```

### ログ表示

```sh
docker compose rogs -f
```
